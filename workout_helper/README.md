# About Our Software
An exercise app for beginners whose hobby is exercising and want help without having to go through the embarassment and costs that may come with having a personal trainer. This app tracks the user's movements and notifies them whenever they are doing the workout correctly or incorrectly. Utilmately, it ensures that the user is doing the workout correctly to avoid muscle strains to enforce proper exercising habits. It gives the user positive feedback which keeps the user happy while exercising so that they will want to come back to their personal cat trainer.

## Commands
1. Clone the repository
    git clone https://github.com/gencsis/wics_hackathon2026.git
    cd wics_hackathon2026
2. Install dependencies
    flutter pub get
3. Run the app
    flutter run
4. Hot reload for quick testing changes
    - Press 'r' in the terminal while the app is running
    - Or save the file in your editor
5. Build the app
    flutter build
